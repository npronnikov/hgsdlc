# Example Flows

This directory contains example flow definitions in Markdown format.

## Structure

Each flow is a `.md` file following the Human-Guided SDLC flow specification.

## Example

```markdown
# My Flow

## Phase 1: Setup

### Node: Initialize
- Type: skill
- Handler: initialize-project

### Gate: Review Setup
- Type: manual-approval
- Required Approvers: 1
```
